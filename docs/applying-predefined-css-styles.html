Applying Predefined CSS Styles


It is possible to assign several visual styles for a certain content type (paragraph, list, image). If this is the case Sitecake will display the list of additional style options in the drop down menu on the toolbar. Style drop down is visible only when there are additional styles for a certain content type. No styles, no selector.
