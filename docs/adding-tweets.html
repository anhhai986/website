Embedding Tweets


Embedded tweets are good for testimonials, for example.

There are two ways to add a tweet :

  • on a Twitter website select a tweet from a feed, click to open a single tweet, drag and drop tweet URL from the address bar of your browser tab onto the page
  • select a tweet from a feed, drag and drop time of a tweet onto the page (yes, time on tweet is a link to that single tweet)
