複数言語対応

Sitecakeでは、インターフェースに使用される言語（アイコンとボタンのラベル、テキストメッセージなど）を切り替えることができます。

現在使用できる言語は以下の通りです：

  • クロアチア語 (hr)
  • チェコ語 (cs)
  • デンマーク語 (dk)
  • オランダ語 (nl)
  • 英語 (en)
  • フランス語 (fr)
  • ドイツ語 (de)
  • イタリア語 (it)
  • ノルウェー語 (nn)
  • ポルトガル語 (pt)
  • ブラジルポルトガル語 (pt-br)
  • ロシア語 (ru)
  • セルビア語 (sr)
  • スロバキア語 (sk)
  • スロベニア語 (sl)
  • スペイン語 (es)
  • トルコ語 (tr)

インタフェース言語については、editor.cnfファイルの LocaleProxyImpl.interfaceLocale設定のパラメータを指定してください。

# interface language (hr, cs, dk, nl, en, fr, de, it, nn, pt, pt-br, ru, sk, sl, es, tr, auto)
LocaleProxyImpl.interfaceLocale = en

対応する言語の2文字 (ISO 639-2言語コード)または、‘auto’（自動）オプションに設定することができます。

自動オプションが設定されている場合は、GETパラメータscln（提示されている場合）またはブラウザ（またはシステム）の言語設定のいずれかによってインタフェース言語が選択されます。

以下は、LocaleProxyImpl.interfaceLocaleが'autoに設定されており、sclnパラメータによって英語をインターフェース言語として選択しているSitecakeページのURLです:

http://example.com/about.html?scln=en
